# Conan Tutorial

## Install Conan
`pip install conan`

Installs conan-2.0.14 on Ubuntu 22

See [Conan Install](https://docs.conan.io/1/installation.html)

## Create a “Hello World” C++ library example project: 
```
conan new cmake_lib -d name=hello -d version=1.0
```
